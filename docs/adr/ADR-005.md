# ADR-005 — Üretim = creative-engine; mockup ≠ prova
**Durum:** KABUL

Baskı-sınıfı üretim çıktıları creative-engine'de (Repo C, bu repo) üretilir. Mockup ayrı çıktı sınıfıdır, ASLA baskı provası yerine geçmez.

*C-repo'daki somut karşılıklar: `export_records.kind` kanal ayrımı (print/preview/mockup/…); F8-D damgası ("MOCKUP — baskı provası değildir", yalnız MockupPage, piksel-gömülü); `MOCKUP_MAX_W=1600` anti-kaçış tavanı (yüksek-çöz mockup yolu bilerek yok). Kaynak: Faz 0 §18 (tam metin styva /docs'ta).*
