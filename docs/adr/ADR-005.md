# ADR-005 — Üretim = creative-engine; mockup ≠ prova
**Durum:** KABUL

Baskı-sınıfı üretim çıktıları creative-engine'de (Repo C, bu repo) üretilir. Mockup ayrı çıktı sınıfıdır, ASLA baskı provası yerine geçmez.

*C-repo'daki somut karşılıklar: `export_records.kind` kanal ayrımı (print/preview/mockup/… — sözlük: docs/export-kinds.md); F8-D damgası ("MOCKUP — baskı provası değildir", yalnız MockupPage, piksel-gömülü); `MOCKUP_MAX_W=1600` anti-kaçış tavanı. Kaynak: Faz 0 §18 (tam metin styva /docs'ta).*

**F8-E tadili:** yüksek-çözünürlük mockup koruması "yolun yokluğu"ndan "kapılı yol"a evrildi — ayrı uç + zorunlu re-onay literal'i ("baskı için değildir") + koşulsuz damga + ayrı export kind (mockup_hires) + MOCKUP_HIRES_MAX_W=3200 tavanı. Varsayılan mockup yolu 1600 tavanıyla AYNEN korunur.
