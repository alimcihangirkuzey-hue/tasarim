# ADR-002 — Korkuluklu Konva
**Durum:** KABUL

Creative Studio editörü (Repo B) Konva üzerine, korkuluklu (guard-railed) desenle kurulur — serbest kanvas değil, kısıtlı/şablon-güdümlü düzenleme.

*Kaynak: Faz 0 §15 (tam metin styva /docs'ta); C-repo etkisi: yok (C'nin editörü ayrı — atölye ürünü).*

**FEDERASYON ŞERHİ (D-29/D-31, 2026-07-17):** Bu ADR'nin ilkesi — korkuluklu Konva, serbest kanvas değil — aynen yürürlüktedir. 'Repo B / Creative Studio' ataması federasyonla değişmiştir: editör TEZGÂH capability'sidir (STYVA Federation Directive v1). 'C-repo etkisi yok' cümlesi geçersizdir.

*P2 CAP-CANVAS-01 somutlaşması: izin-listeli araç seti (`CANVAS_TOOLS`) · sahne-sınırı kilidi (`clampToBounds`) · ızgara (`snapToGrid`) · min-boyut + döndürmesiz kısıtlı Transformer · zoom tavan/tabanı (`zoomAt`) — tümü saf fonksiyon olarak `packages/shared/src/canvas.ts`'te, tüm mutasyonlar `canvasReduce` tek kapısından.*
