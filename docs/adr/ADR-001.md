# ADR-001 — styva çatı
**Durum:** KABUL · **Tadil:** D-9 (üç yaşayan ürün + contract entegrasyonu)

styva = SaaS platform çatısı (Repo B). D-9 tadiliyle: üç repo (A SWISS · B styva · C TEZGÂH) yaşayan ürünlerdir; hiçbiri pasifleştirilemez/eritilemez; entegrasyon yalnız sürümlü Shared Contract'larla (Render · Menu/QR · Tenant/Identity).

*Kaynak: Faz 0 §14 (tam metin styva /docs'ta); bu dosya kokpit v14 §3 endeksinden C-repo özeti.*
